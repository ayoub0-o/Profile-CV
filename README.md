# Personal CV Website

A responsive personal CV website built with HTML, CSS, and JavaScript, deployable on Vercel.

## Features

- Responsive design
- Smooth scrolling navigation
- Interactive skill cards
- Contact form with serverless backend
- Modern gradient design

## Deployment on Vercel

1. Push this code to a GitHub repository
2. Connect your GitHub repo to Vercel
3. Deploy automatically

## Contact Form

The contact form uses a serverless function to handle submissions. You can integrate it with email services like SendGrid or Mailgun for actual email delivery.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Vercel (for deployment and serverless functions)
